# Sasha's Wiki Mirror 🐦

This repository is a public mirror of select content from [atigerseye.com/wiki](https://atigerseye.com/wiki). 
It is automatically updated every hour from the live DokuWiki instance.

## What's in here?
* **/pages**: The raw markdown/DokuWiki text for public wiki articles.
  * *Note: Restricted/private folders are strictly excluded.*
* **/plugins**: Custom DokuWiki plugins, including **yzruby** (a Yivalese/rubytext syntax handler, WIP).

## About
This mirror allows for version tracking of public wiki content and sharing of custom DokuWiki extensions developed for the Yivalese project.
